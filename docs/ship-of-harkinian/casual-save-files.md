# Casual Save Files

Saves for every dungeon of the game, for the people who have lost their save file for any reason and don't want to mess around with the flags and the save editor which let's be honest its a mess to make it work properly if you are playing casually and don't know what are you doing.

All the saves are made from a casual playthough of the game: No cheats, no glitches, no skips, no shenanigans of any kind, all the flags were reached 100% naturally. Also worth mentioning that I intentionally didn't pick any Piece of Heart, Skulltula, or upgrade that isn't requiered except for Biggoron Sword, so each player can do those quests if they want.

HOW TO USE: With the game closed, download the save file you need from the table below, place it in the `/saves/` folder and rename it `file#.sav` (replacing the # with either 1, 2 or 3 depending on which slot you want the save to load) and then run the game as normal, EZ.

## Additional Information

- From Forest Temple and until Ganondorf's Castle, on each boss save there is a Farore's Wind set up on the closest door to the boss.
- Shadow Temple comes before Spirit Temple, as always should be.
- First half of Biggoron Sword trade sequence was made after Forest Temple (until Poacher's Saw) and the second one after Shadow Temple (save #25 has Biggoron Sword)
- Only missing main items are Magic Beans and two Bottles.
- On the 100% save only missing thing is the Malon race because since I was a kid I always thought Link's house was too small to have a cow in there, so I never get it on my hundo playthroughs.
