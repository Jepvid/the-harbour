# Mod Development

Welcome to the Ship of Harkinian mod development guide! This section contains everything you need to create custom content for Ship of Harkinian.

## Getting Started

Ship of Harkinian supports a wide variety of mods, from simple texture replacements to complex custom models, animations, and even scene modifications. The modding system uses OTR (OoT Resource) archives that package your custom assets.

### What You Can Mod

- **Textures** - Replace or enhance any texture in the game
- **Models** - Create custom player models, enemies, items, and objects
- **Animations** - Add or modify character animations
- **Audio** - Replace sound effects and voice samples
- **Scenes** - Modify level geometry and environments
- **And more!** - Equipment, UI elements, effects...

### Prerequisites

To create mods, you'll need:
- **Blender** (3.0 or newer recommended)
- **Fast64 Plugin** - The primary tool for exporting N64-format assets
- **Basic 3D modeling knowledge** - Understanding of meshes, materials, and UV mapping
- **A legally obtained ROM** - Required for extracting reference assets

## Quick Start Path

**For Beginners:**
1. Read through [Modding Tips](./modding-tips.md) to understand requirements
2. Watch the setup tutorial in [Tutorials](./tutorials.md)
3. Download Fast64 from [Tools and Resources](./tools-and-resources.md)
4. Start with a simple texture replacement
5. Progress to model replacements

**For Experienced Modders:**
1. Grab Fast64 and reference the [Tools and Resources](./tools-and-resources.md)
2. Check [Modding Tips](./modding-tips.md) for Ship-specific requirements
3. Use the [Technical Guide](./technical-guide.md) as needed
4. Join the Discord for support
