# Compatible Glitches

This is intended to be the list of all glitches that are either incompatible with 2Ship entirely, or incompatible with specific enhancements. If a glitch is not present on this list then you can assume it is fully compatible with 2ship, if you find otherwise please let us know so we can update this list.

## Fully Incompatible:
- All SRM related glitches - None of them have been tested to my knowledge, but the general assumption is none of them will work properly.
- [Ocarina Dive](https://www.zeldaspeedruns.com/mm/tech/ocarina-dive) - Methods 1 and 2 are not working, 3 and 4 are working. We would like to fix 2 but haven't had much luck yet. Help wanted.
- [Bottle Adventure](https://www.zeldaspeedruns.com/mm/misc/bottle-adventure) - Untested, but likely to share the same issues with SoH that we later fixed (Endianness being wrong)

## Incompatible w/wo Enhancements:
- [Song storage](https://www.zeldaspeedruns.com/mm/tech/song-storage) - Incompatible with `Faster Song Playback`, as you are supposed to void out during the playback that is now skipped
- [Pause Buffering](https://www.zeldaspeedruns.com/mm/tech/pause-buffer) - By default the window for buffering is 0 frames, but it can be increased (`Enhancements > Restorations > Pause Buffer Input Window`)
- [Power Crouch Stab](https://www.zeldaspeedruns.com/mm/tech/power-crouch-stab) - Patched by default, as it was in US/EU roms, but unpatchable (`Enhancements > Restorations > Power Crouch Stab`)

## Compatible
- Weirdshots
- 4th day (Telescope method)
- [0th day](https://www.zeldaspeedruns.com/mm/misc/0th-day)
- [Mayor's Warp](https://www.zeldaspeedruns.com/mm/warpglitches/mayors-warps)
- Any other glitch you don't see above. If something is missing please let us know in this thread.
